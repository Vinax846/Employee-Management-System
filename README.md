Project Description: 

Employee Management System is a desktop-based application developed using Java (JFrame) and SQL. It helps organizations manage employee records efficiently by providing features to add, view, update, delete, and search employee details.

Features:

➕ Add new employee details
📄 View all employee records
✏️ Update existing employee information
❌ Remove employee data
🔍 Search employees by ID or name
🗂️ Organized database management using SQL

Technologies Used: 

Java (JFrame) – GUI development
SQL (MySQL) – Database management
JDBC – Connectivity between Java and database
